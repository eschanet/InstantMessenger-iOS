Detailed Description

FBNCardViewController is the main entry point for presenting a card that represents In-App Notification from a payload. This class encapsulates the display and layout of the entire push card and is intended to be presented full screen.

It is invalid to initialize this class yourself, but it's intended to be used from FBNotificationsManager.
