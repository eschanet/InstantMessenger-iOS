\subsection{Detailed Description}
The {\ttfamily \hyperlink{interface_parse_client_configuration}{Parse\+Client\+Configuration}} represents the local configuration of the S\+D\+K to connect to the server with.

These configurations can be stored, copied, and compared, but cannot be safely changed once the S\+D\+K is initialized.

Use this object to construct a configuration for the S\+D\+K in your application, and pass it to {\ttfamily \hyperlink{interface_parse}{Parse}.+initialize\+With\+Configuration\+:}.
