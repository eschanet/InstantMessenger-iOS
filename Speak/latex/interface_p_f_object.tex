Saves this object to the server at some unspecified time in the future, even if \hyperlink{interface_parse}{Parse} is currently inaccessible.

Use this when you may not have a solid network connection, and don\textquotesingle{}t need to know when the save completes. If there is some problem with the object such that it can\textquotesingle{}t be saved, it will be silently discarded.

Objects saved with this method will be stored locally in an on-\/disk cache until they can be delivered to \hyperlink{interface_parse}{Parse}. They will be sent immediately if possible. Otherwise, they will be sent the next time a network connection is available. Objects saved this way will persist even after the app is closed, in which case they will be sent the next time the app is opened. If more than 10\+M\+B of data is waiting to be sent, subsequent calls to {\ttfamily -\/save\+Eventually} will cause old saves to be silently discarded until the connection can be re-\/established, and the queued objects can be saved.

Saves this object to the server at some unspecified time in the future, even if \hyperlink{interface_parse}{Parse} is currently inaccessible.

Use this when you may not have a solid network connection, and don\textquotesingle{}t need to know when the save completes. If there is some problem with the object such that it can\textquotesingle{}t be saved, it will be silently discarded. If the save completes successfully while the object is still in memory, then callback will be called.

Objects saved with this method will be stored locally in an on-\/disk cache until they can be delivered to \hyperlink{interface_parse}{Parse}. They will be sent immediately if possible. Otherwise, they will be sent the next time a network connection is available. Objects saved this way will persist even after the app is closed, in which case they will be sent the next time the app is opened. If more than 10\+M\+B of data is waiting to be sent, subsequent calls to {\ttfamily -\/save\+Eventually\+:} will cause old saves to be silently discarded until the connection can be re-\/established, and the queued objects can be saved.
