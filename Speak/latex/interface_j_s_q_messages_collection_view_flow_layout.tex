\subsection{Detailed Description}
The {\ttfamily \hyperlink{interface_j_s_q_messages_collection_view_flow_layout}{J\+S\+Q\+Messages\+Collection\+View\+Flow\+Layout}} is a concrete layout object that inherits from {\ttfamily U\+I\+Collection\+View\+Flow\+Layout} and organizes message items in a vertical list. Each {\ttfamily \hyperlink{interface_j_s_q_messages_collection_view_cell}{J\+S\+Q\+Messages\+Collection\+View\+Cell}} in the layout can display messages of arbitrary sizes and avatar images, as well as metadata such as a timestamp and sender. You can easily customize the layout via its properties or its delegate methods defined in {\ttfamily \hyperlink{protocol_j_s_q_messages_collection_view_delegate_flow_layout-p}{J\+S\+Q\+Messages\+Collection\+View\+Delegate\+Flow\+Layout}}.
