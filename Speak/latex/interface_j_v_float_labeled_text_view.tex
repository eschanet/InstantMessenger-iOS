\subsection{Detailed Description}
{\ttfamily \hyperlink{interface_j_v_float_labeled_text_view}{J\+V\+Float\+Labeled\+Text\+View}} is a {\ttfamily U\+I\+Text\+View} subclass that implements the \char`\"{}\+Float Label Pattern\char`\"{}.

Due to space constraints on mobile devices, it is common to rely solely on placeholders as a means to label fields. This presents a U\+X problem, in that, once the user begins to fill out a form, no labels are present.

{\ttfamily \hyperlink{interface_j_v_float_labeled_text_view}{J\+V\+Float\+Labeled\+Text\+View}} aims to improve the user experience by having placeholders transition into \char`\"{}floating labels\char`\"{} that hover above the text view after it is populated with text.

\hyperlink{interface_j_v_float_labeled_text_view}{J\+V\+Float\+Labeled\+Text\+View} supports i\+O\+S 7+.

Credits for the concept to Matt D. Smith (), and his original design\+: \href{http://mattdsmith.com/float-label-pattern/}{\tt http\+://mattdsmith.\+com/float-\/label-\/pattern/}
