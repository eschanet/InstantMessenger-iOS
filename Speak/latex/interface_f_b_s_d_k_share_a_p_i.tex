\subsection{Detailed Description}
A utility class for sharing through the graph A\+P\+I. Using this class requires an access token in \mbox{[}\hyperlink{interface_f_b_s_d_k_access_token}{F\+B\+S\+D\+K\+Access\+Token} current\+Access\+Token\mbox{]} that has been granted the \char`\"{}publish\+\_\+actions\char`\"{} permission.  \hyperlink{interface_f_b_s_d_k_share_a_p_i}{F\+B\+S\+D\+K\+Share\+A\+P\+I} network requests are scheduled on the current run loop in the default run loop mode (like N\+S\+U\+R\+L\+Connection). If you want to use \hyperlink{interface_f_b_s_d_k_share_a_p_i}{F\+B\+S\+D\+K\+Share\+A\+P\+I} in a background thread, you must manage the run loop yourself.
