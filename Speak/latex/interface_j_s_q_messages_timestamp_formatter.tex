\subsection{Detailed Description}
An instance of {\ttfamily \hyperlink{interface_j_s_q_messages_timestamp_formatter}{J\+S\+Q\+Messages\+Timestamp\+Formatter}} is a singleton object that provides an efficient means for creating attributed and non-\/attributed string representations of {\ttfamily N\+S\+Date} objects. It is intended to be used as the method by which you display timestamps in a {\ttfamily \hyperlink{interface_j_s_q_messages_collection_view}{J\+S\+Q\+Messages\+Collection\+View}}.
