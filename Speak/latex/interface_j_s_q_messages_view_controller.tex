\subsection{Detailed Description}
The {\ttfamily \hyperlink{interface_j_s_q_messages_view_controller}{J\+S\+Q\+Messages\+View\+Controller}} class is an abstract class that represents a view controller whose content consists of a {\ttfamily \hyperlink{interface_j_s_q_messages_collection_view}{J\+S\+Q\+Messages\+Collection\+View}} and {\ttfamily \hyperlink{interface_j_s_q_messages_input_toolbar}{J\+S\+Q\+Messages\+Input\+Toolbar}} and is specialized to display a messaging interface.

\begin{DoxyWarning}{Warning}
This class is intended to be subclassed. You should not use it directly. 
\end{DoxyWarning}
