\subsection{Detailed Description}
Class which represents the metadata on an object in Firebase Storage. This metadata is returned on successful operations, and can be used to retrieve download U\+R\+Ls, content types, and a \hyperlink{interface_f_i_r_storage}{F\+I\+R\+Storage} reference to the object in question. Full documentation can be found at the G\+C\+S Objects\+::resource docs. \begin{DoxySeeAlso}{See also}
\href{https://cloud.google.com/storage/docs/json_api/v1/objects#resource}{\tt https\+://cloud.\+google.\+com/storage/docs/json\+\_\+api/v1/objects\#resource} 
\end{DoxySeeAlso}
