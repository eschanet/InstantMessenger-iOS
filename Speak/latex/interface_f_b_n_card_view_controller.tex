\subsection{Detailed Description}
{\ttfamily \hyperlink{interface_f_b_n_card_view_controller}{F\+B\+N\+Card\+View\+Controller}} is the main entry point for presenting a card that represents In-\/\+App Notification from a payload. This class encapsulates the display and layout of the entire push card and is intended to be presented full screen.

It is invalid to initialize this class yourself, but it\textquotesingle{}s intended to be used from {\ttfamily \hyperlink{interface_f_b_notifications_manager}{F\+B\+Notifications\+Manager}}.
