\item Simple comparisons such as {\ttfamily =}, {\ttfamily !=}, {\ttfamily $<$}, {\ttfamily $>$}, {\ttfamily $<$=}, {\ttfamily $>$=}, and {\ttfamily B\+E\+T\+W\+E\+E\+N} with a key and a constant.
\item Containment predicates, such as {\ttfamily x I\+N \{1, 2, 3\}}.
\item Key-\/existence predicates, such as {\ttfamily x I\+N S\+E\+L\+F}.
\item B\+E\+G\+I\+N\+S\+W\+I\+T\+H expressions.
\item Compound predicates with {\ttfamily A\+N\+D}, {\ttfamily O\+R}, and {\ttfamily N\+O\+T}.
\item Sub\+Queries with {\ttfamily key I\+N \%@}, subquery.
\end{DoxyItemize}

The following types of predicates are N\+O\+T supported\+:


\begin{DoxyItemize}
\item Aggregate operations, such as {\ttfamily A\+N\+Y}, {\ttfamily S\+O\+M\+E}, {\ttfamily A\+L\+L}, or {\ttfamily N\+O\+N\+E}.
\item Regular expressions, such as {\ttfamily L\+I\+K\+E}, {\ttfamily M\+A\+T\+C\+H\+E\+S}, {\ttfamily C\+O\+N\+T\+A\+I\+N\+S}, or {\ttfamily E\+N\+D\+S\+W\+I\+T\+H}.

\item Complex predicates with many O\+Red clauses.
