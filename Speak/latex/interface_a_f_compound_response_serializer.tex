\subsection{Detailed Description}
{\ttfamily A\+F\+Compound\+Serializer} is a subclass of {\ttfamily \hyperlink{interface_a_f_h_t_t_p_response_serializer}{A\+F\+H\+T\+T\+P\+Response\+Serializer}} that delegates the response serialization to the first {\ttfamily \hyperlink{interface_a_f_h_t_t_p_response_serializer}{A\+F\+H\+T\+T\+P\+Response\+Serializer}} object that returns an object for {\ttfamily response\+Object\+For\+Response\+:data\+:error\+:}, falling back on the default behavior of {\ttfamily \hyperlink{interface_a_f_h_t_t_p_response_serializer}{A\+F\+H\+T\+T\+P\+Response\+Serializer}}. This is useful for supporting multiple potential types and structures of server responses with a single serializer.
