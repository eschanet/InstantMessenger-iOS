\subsection{Detailed Description}
The {\ttfamily \hyperlink{protocol_j_s_q_message_data-p}{J\+S\+Q\+Message\+Data}} protocol defines the common interface through which a {\ttfamily \hyperlink{interface_j_s_q_messages_view_controller}{J\+S\+Q\+Messages\+View\+Controller}} and {\ttfamily \hyperlink{interface_j_s_q_messages_collection_view}{J\+S\+Q\+Messages\+Collection\+View}} interact with message model objects.

It declares the required and optional methods that a class must implement so that instances of that class can be displayed properly within a {\ttfamily \hyperlink{interface_j_s_q_messages_collection_view_cell}{J\+S\+Q\+Messages\+Collection\+View\+Cell}}.

Two concrete classes that conform to this protocol are provided in the library. See {\ttfamily J\+S\+Q\+Text\+Message} and {\ttfamily J\+S\+Q\+Media\+Message}.

This method is used to determine if the message data item contains text or media. If this method returns {\ttfamily Y\+E\+S}, an instance of {\ttfamily \hyperlink{interface_j_s_q_messages_view_controller}{J\+S\+Q\+Messages\+View\+Controller}} will ignore the {\ttfamily text} method of this protocol when dequeuing a {\ttfamily \hyperlink{interface_j_s_q_messages_collection_view_cell}{J\+S\+Q\+Messages\+Collection\+View\+Cell}} and only call the {\ttfamily media} method.

Similarly, if this method returns {\ttfamily N\+O} then the {\ttfamily media} method will be ignored and and only the {\ttfamily text} method will be called.

\begin{DoxyReturn}{Returns}
A boolean value specifying whether or not this is a media message or a text message. Return {\ttfamily Y\+E\+S} if this item is a media message, and {\ttfamily N\+O} if it is a text message.
