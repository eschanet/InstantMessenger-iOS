\subsection{Detailed Description}
The {\ttfamily \hyperlink{interface_p_f_a_c_l}{P\+F\+A\+C\+L}} class is used to control which users can access or modify a particular object. Each {\ttfamily \hyperlink{interface_p_f_object}{P\+F\+Object}} can have its own {\ttfamily \hyperlink{interface_p_f_a_c_l}{P\+F\+A\+C\+L}}. You can grant read and write permissions separately to specific users, to groups of users that belong to roles, or you can grant permissions to \char`\"{}the public\char`\"{} so that, for example, any user could read a particular object but only a particular set of users could write to that object.
