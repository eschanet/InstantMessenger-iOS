\subsection{Detailed Description}
{\ttfamily \hyperlink{interface_p_f_geo_point}{P\+F\+Geo\+Point}} may be used to embed a latitude / longitude point as the value for a key in a {\ttfamily \hyperlink{interface_p_f_object}{P\+F\+Object}}. It could be used to perform queries in a geospatial manner using {\ttfamily \hyperlink{class_p_f_query}{P\+F\+Query}.-\/where\+Key\+:near\+Geo\+Point\+:}.

Currently, instances of {\ttfamily \hyperlink{interface_p_f_object}{P\+F\+Object}} may only have one key associated with a {\ttfamily \hyperlink{interface_p_f_geo_point}{P\+F\+Geo\+Point}} type.
