\subsection{Detailed Description}
\hyperlink{interface_b_f_cancellation_token_source}{B\+F\+Cancellation\+Token\+Source} represents the producer side of a Cancellation\+Token. Signals to a Cancellation\+Token that it should be canceled. It is a cancellation token that also has methods for changing the state of a token by cancelling it.
