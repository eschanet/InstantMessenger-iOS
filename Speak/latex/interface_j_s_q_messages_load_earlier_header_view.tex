\subsection{Detailed Description}
The {\ttfamily \hyperlink{interface_j_s_q_messages_load_earlier_header_view}{J\+S\+Q\+Messages\+Load\+Earlier\+Header\+View}} class implements a reusable view that can be placed at the top of a {\ttfamily \hyperlink{interface_j_s_q_messages_collection_view}{J\+S\+Q\+Messages\+Collection\+View}}. This view contains a \char`\"{}load earlier messages\char`\"{} button and can be used as a way for the user to load previously sent messages.
