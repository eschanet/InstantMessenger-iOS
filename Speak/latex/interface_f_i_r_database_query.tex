\subsection{Detailed Description}
A \hyperlink{interface_f_i_r_database_query}{F\+I\+R\+Database\+Query} instance represents a query over the data at a particular location.

You create one by calling one of the query methods (query\+Ordered\+By\+Child\+:, query\+Starting\+At\+Value\+:, etc.) on a \hyperlink{interface_f_i_r_database_reference}{F\+I\+R\+Database\+Reference}. The query methods can be chained to further specify the data you are interested in observing
